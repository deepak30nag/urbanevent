Git Repo for Urban clap demo simulating microservices concepts in java using spring boot framework.
Mainly divided into two services:
1. order service 
2. user service
3. event service

Cross cutting concerns
1. server registry and discovery
2. Api Gateway

PORT configuration
1. order service => 8100 onwards
2. event service => 8200 onwards
3. user service => 8080 onwards
4. server registry => 8761
5. api gateway => 8765
